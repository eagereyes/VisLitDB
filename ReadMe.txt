This repository contains some data and code to explore visualization publication data.

db:
	Setup scripts for a relational database of citation data.

statistics:
	Submissions, number of accepted papers, and acceptance rates for a number
	of visualization conferences. Data is available in CSV and JSON formats,
	and there is a Protovis-based visualization of the data.
	See also: http://eagereyes.org/service/acceptance-rates
	
LiteratureDB:
	Java code to move citation data into a neo4j graph database.
